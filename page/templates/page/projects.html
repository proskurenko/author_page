Проект #1

Исследование и разработка технологических вариантов внепечной обработки стали и разливка стали на МНЛЗ сверхнизкоуглеродистых марок сталей (IF) с целью обеспечения их необходимой чистоты.

Личный вклад в проект

Целью проекта является разработка технологических рекомендаций для оптимизации схемы производства стали класса IF.

Личное участие состояло в исследовании поведения межфазной поверхности "шлак-металл" при перемешивании расплава в агрегате ковш-печь (АКП).

Это достигалось путем лабораторного исследования отобранных проб металла и шлака на предмет изучения формы металличекой дисперсии металлической фазы в шлаке. О некоторых результатах этого исследования можно узнать в опубликованной статье.

По результатам исследования также был получен патент на полезную модель.

Проект #2

Разработка состава уплотнительной вставки между погружным стаканом и стаканом-дозатором МНЛЗ для снижения вторичного окисления стали в процессе непрерывной разливки.

Личный вклад в проект

Цель проекта - предложить к изготовлению для практического применения состав уплотнительной вставки из эластичного огнеупорного материала, который бы при его применении в месте между погружным стаканом и стаканом-дозатором МНЛЗ обеспечивал бы герметичную их состыковку, в течение всей разливки.

Личное участие состояло именно в лабораторном исследовании образцов различного композиционного огнеупорного состава с целью предложения нужного для изготовления изделия.

Результатом исследования стало промышленное изготовление и успешное внедрение.

С графической информацией этого проекта можно ознакомиться из портфолио

Проект #3

Разработка состава огнеупорной саморазрушающейся пробки, устанавливаемой в летке конвертера, для отсечки шлака при выпуске стали в ковш.

Личный вклад в проект

Цель проекта - дать практические рекомендации по составу пластичного огнеупорного материала, который бы надежно закупоривал отверстие летки конвертера, но в момент выпуска, легко разрушался бы при контакте с ванной жидкой стали в момент опрокидывания ковертера.

Личное участие состояло в подборе композитного состава такого огнеупора, который бы саморазрушался в нужный момент. Для этого нужно было испытать в лабораторных условиях необходимое количестко образцов проектного состава, нужное количество именно для получения требуемого. По результатам исследования был предложен оптимальный состав для практического испытания.

К сожалению, дальше лабораторных испытаний, проект развития не получил.

С графической информацией этого проекта частично можно ознакомиться из портфолио

Проект #4

Рафинирование металла в промежуточном ковше МНЛЗ с целью повышения чистоты стали и снижения зарастания погружного стакана.

Личный вклад в проект

Цель проекта - разработать технологию повышения чистоты стали путем частичного рафинирования ее в промежуточном ковше МНЛЗ.

Личное участие состояло в изучении структры зарастания погружного стакана, исследования механизма этого явления. По результатам исследования были предложены запатентованные устройство для рафинирования металлического распава и способ .

С графической информацией этого проекта частично можно ознакомиться из портфолио

Проект #5

Проектирование основного плавильного оборудования цеха.

Личный вклад в проект

Цель проекта - спроектировать необходимое оборудование для плавильно-литейного цеха, где основным плавильным агрегатом должна быть индукционная тигальная печь.

Моим заданием было именно расчёт основых параметров индукционный тигельной печи. Результат задания был представлен в виде, разработанной на языке программирования Delphi 7, небольшой программы, которая учитывает все необходимые технологические параметры.
